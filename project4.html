Model 18 Beechcraft Stability Analysis

Project Overview

This project focused on conducting comprehensive stick-fixed stability analysis for the Beechcraft Model 18 AT-11. The project included developing a versatile Excel spreadsheet and MATLAB code to verify the accuracy of the calculations and estimate performance parameters.

Key Contributions

  • Performed full stick-fixed longitudinal, directional, and lateral stability analysis.
  • Developed an Excel spreadsheet and MATLAB code to validate the results.
  • Collaborated with a team of four to optimize the stability performance of the aircraft.
